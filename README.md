# 🏦 Banque UY1 - Application Bancaire

## Description
Application bancaire complète avec backend Node.js et frontend Vue.js.

## Fonctionnalités
- Authentification (JWT)
- Gestion des comptes (CHECKING/SAVINGS)
- Dépôt externe
- Retrait
- Transfert entre comptes
- Virement externe
- Panel Admin

## Stack Technique
- **Backend**: Node.js, Express, SQLite, JWT
- **Frontend**: Vue.js, Vite, Pinia
- **Tests**: Vitest
- **Déploiement**: Render

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
